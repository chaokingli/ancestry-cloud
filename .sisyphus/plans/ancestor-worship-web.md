# 拜祭先祖网页 (Ancestry Cloud) - 完整计划

## TL;DR

> **Quick Summary**: 创建一个家族共享的拜祭先祖网页应用，支持先祖照片管理、致词功能、虚拟祭拜（上香、供奉、磕头/作揖、烧纸钱），采用水墨风格设计，响应式支持桌面/平板/手机。

### 技术栈
- **前端**: Next.js 16 + TypeScript + Tailwind CSS + shadcn/ui

- **认证**: NextAuth.js + bcrypt
- **样式**: 水墨风格设计系统
- **测试**: Vitest + Playwright
- **部署**: Docker
